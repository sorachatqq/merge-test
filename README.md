# Merge Arrays

## Tests

Unit tests:
- merge three arrays correctly
- handle empty arrays
- handle all values from empty collections
- handle arrays with duplicate value
- should handle negative numbers

## การติดตั้ง Dependencies

### ใช้ npm:

```bash
npm install
```

### ใช้ yarn:

```bash
yarn install
```

## การ Build โปรเจกต์

### ใช้ npm:

```bash
npm run build
```

### ใช้ yarn:

```bash
yarn build
```

## การรัน Unit Tests

### ใช้ npm:

```bash
# รัน tests ทั้งหมด
npm test

# รัน tests ในโหมด watch
npm run test:watch

# รัน tests พร้อม coverage report
npm run test:coverage
```

### ใช้ yarn:

```bash
# รัน tests ทั้งหมด
yarn test

# รัน tests ในโหมด watch (auto-rerun เมื่อไฟล์เปลี่ยน)
yarn test:watch

# รัน tests พร้อม coverage report
yarn test:coverage
```
